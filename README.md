# Untac-Rx web

```
2021.9.5 ~ 2021.10.8
```

## Project Stack

- **React**
- React router dom
- Typescript
- Recoil
- React query
- Emotion

## Developing

Runs the app in the development mode:

```bash
yarn start
```

## Building

Builds the app for production to the `build` folder:

```bash
yarn build
```
